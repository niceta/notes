Заголовок и текстовое поле - text field и text view соответственно, тут все очевидно

Штуку для выбора даты вынес в отдельный xib - DatePickerView, содержит свич, лейбл и текстовое поле, в которое пишется выбранная дата

4 квадратика для выбора цвета лежат в stack view, каждый квадратик относится к классу
ColorSquareView. В этом же классе лежит функция draw, которая отвечает за отрисовку
галочки выбора для квадратика: рисовал с помощью UIBezierPath

ColorPicker вынесен в отдельный xib, спасибо stack overflow за него =)
Активируется, как и задание требует, лонгтапом по крайнему квадратику с градиентом
Сделал его по совету из комментариев просто отдельной вьюхой, которая по умолчанию
невидима и просто показывается или же скрывается

1й клик по квадратику с колопикером - только лонгтап, чтобы не пропала вьюшка с галочкой
После лонгтапа уже можно будет на 4й квадратик ставить галочку

Заранее спасибо за справедливые замечания ! =)
